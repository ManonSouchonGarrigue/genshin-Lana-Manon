# genshin-Lana-Manon

Nom du Projet : Genshin Impact

Genshin Impact est un jeu vidéo d'action-RPG en monde ouvert. Nous avons donc décidé de créer un site web consacré à ce jeu afin d’aider les joueurs à suivre les mises à jour, découvrir les nouveaux personnages et se tenir informés des nouvelles histoires et événements du jeu.

Technologies utilisées : html, css, js, json.

Page du site : 
- Home : ppage présentant les nouveautés du jeu ainsi qu’une courte description de l’univers de Genshin Impact, avec également un chatbot nommé Catherine qui répond aux questions des utilisateurs.
- Personnages : page dédiée aux personnages du jeu, avec des filtres permettant de les trier par niveau, élément, région et arme utilisée.
- Armes : page regroupant les différentes armes du jeu, filtrables par niveau et type d’arme.
- Quiz : page interactive permettant de tester ses connaissances sur le jeu, avec plusieurs tentatives possibles et un système de score.
- About us : page présentant les membres du projet ainsi que les tâches réalisées par chacun.
